Desenvolvi um sistema bancário em Python utilizando Programação Orientada a Objetos, 
com separação modular de responsabilidades. O projeto implementa operações 
como criação de contas,depósitos, saques, transferências e histórico de transações.
Os dados são persistidos em arquivo JSON, aplicando conceitos de serialização e desserialização.
Também utilizei tratamento de exceções e validação de regras de negócio 
(como saldo insuficiente e valores inválidos), garantindo maior robustez e integridade do sistema.
